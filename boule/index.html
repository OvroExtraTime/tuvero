Schönes Spiel!

Unerwarteter Fehler beim Laden. Typische Ursache sind fehlerhafte oder inkompatible Spielstände, die von alten Versionen erstellt wurden. Siehe dazu auch https://boulesprog.wordpress.com/ für Informationen über die letzten Updates. Um die gespeicherten Spielstände zu löschen, kann man #reset öffnen. Sollte der Fehler dann immer noch bestehen, bitte ich um eine Nachricht an erik.e.lorenz@gmail.com.

Konnte eine Skriptdatei nicht laden. Das kann passieren, wenn die Internetverbindung beim Laden der Seite verloren geht und sollte in einer Release-Version nicht vorkommen. Alternativ kann ein Serverausfall dieses Problem verursacht haben. Sollte der Fehler bestehen bleiben, bitte ich um eine Nachricht an erik.e.lorenz@gmail.com.

Erste Schritte

Um ein Turnier durchzuführen, folge diesen Schritten:
  1. Trage die Spieler im Anmeldungen-Tab ein
  2. Wähle und starte ein Turniersystem im Turniersysteme-Tab
  3. Trage die Spielergebnisse im Laufende Spiele-Tab ein
  4. Sieh die Platzierung im Platzierungen-Tab ein
  5. Exportiere die Turnierergebnisse über das Optionen-Tab
Hilfe und Links: Über...-Tab

Warum Chrome?

Zur Nutzung von Tuvero wird der Chrome Browser empfohlen.

Der Chrome Browser bietet eine Druckvorschau, mit der sich Seitenränder etc. anpassen lassen.

Für eine unverbindliche Übersicht aller unterstützten Browser, siehe Über...-Tab

Optionen und Einstellungen

Schriftgröße

Legt auch die Schriftgröße beim Drucken fest.

Dieser Browser kann keine lokalen Dateien laden. Aktualisieren sie ihren Browser oder öffnen sie diese Seite in einem moderneren Browser. Manche Hersteller stellen keine Updates für ihre Browser unter einigen Betriebssystemen zur Verfügung (z.B. Apple mit Safari für Windows).

Tabellen-Export

Excel-kompatible .csv-Dateien exportieren

Damit die Datei geöffnet werden kann, muss sie mit der .csv-Dateiendung gespeichert werden, beispielsweise als 'tuvero.csv'.

ACHTUNG: Dieser Browser unterstützt wichtige Standards nicht, die zum Speichern und Laden notwendig sind. Beim Schließen des Browserfensters gehen sämtliche Daten verloren.

Browserunterstützung

Für diese Software wird auf freie Webstandards gesetzt. Leider werden diese nicht von allen Browsern vollständig unterstützt. Deshalb kann es vorkommen, dass einzelne Funktionen mit einigen Browsern nicht zur Verfügung stehen. Um das nach Möglichkeit zu vermeiden, wird das Programm mit diesen Browsern getestet:

Desktop:

Mobil:

Dieser Browser unterstützt folgende Funktionen nicht:

  • JavaScript
  • HashChange
  • LocalStorage
  • WebWorkers
  • Download Attribute
  • FileReader
  • JSON
