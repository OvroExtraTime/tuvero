Speicherstand kann nicht geladen werden.

Das im Browser gespeicherte Turnier konnte nicht geladen werden. Die Ursache kann in einem Fehler in Tuvero, einem extrem alten Speicherstand oder der falschen Tuvero-Variante liegen. Der Speicherstand wurde nicht überschrieben. Es gibt jetzt folgende Möglichkeiten:
  • Vielleicht wird im Hintergrund schon eine neue Version von Tuvero heruntergeladen, mit der der Speicherstand geöffnet werden kann. Wenn eine neue Version heruntergeladen wurde, kann man sie durch Neuladen starten.
  • Offline-Versionen erhalten keine Updates. Auf https://tuvero.de/ ist immer die neueste Version verfügbar.
  • Wurde die richtige Tuvero-Variante geöffnet? Eine Übersicht ist auf https://tuvero.de/ verfügbar.
  • Wenn der Turnierstand nicht mehr benötigt wird, kann er gelöscht werden. Zum Löschen muss #reset geöffnet werden.
  • Für technische Details kann man die JavaScript-Konsole öffnen (Chrome: Strg+Shift+J). Dort stehen mögliche Ursachen.
  • Wenn der Fehler weiterhin besteht, dann bitte ich um Kontakt, z.B. über erik.e.lorenz@gmail.com. Bitte fügt auch die Ausgabe der JavaScript-Konsole bei.
Informationen zu neuen Versionen und behobenen Fehlern stehen auf http://blog.tuvero.de.

Konnte eine Skriptdatei nicht laden. Sollte der Fehler auch bei wiederholtem Neuladen der Seite bestehen bleiben, bitte ich um eine Nachricht an erik.e.lorenz@gmail.com.

Erste Schritte

Um ein Turnier durchzuführen, folge diesen Schritten:
  1. Trage die Spieler im Anmeldungen-Tab ein
  2. Wähle und starte ein Turniersystem im Turniersysteme-Tab
  3. Trage die Spielergebnisse im Laufende Spiele-Tab ein
  4. Sieh die Platzierung im Platzierungen-Tab ein
  5. Exportiere die Turnierergebnisse über das Optionen-Tab
Hilfe und Links: Über...-Tab

Warum Chrome?

Zur Nutzung von Tuvero wird der Chrome Browser empfohlen.

Der Chrome Browser bietet eine Druckvorschau, mit der sich Seitenränder etc. anpassen lassen.

Für eine unverbindliche Übersicht aller unterstützten Browser, siehe Über...-Tab

Hinweis - Klick mich

Diese Kästen können durch einen Klick auf ihre Titelleiste versteckt und wiederhergestellt werden.
Versteckte Kästen werden nicht gedruckt.
Die Schriftgröße beim Drucken kann im Optionen-Tab eingestellt werden.

Beteiligung an den laufenden Kosten

Tuvero ist für uns neben dem Arbeitsaufwand mit jährlichen Kosten verbunden, die Server, Domänen, Sicherheitszertifikate, Testumgebungen und Archivspeicher beinhalten. Wenn euch Tuvero gefällt und die Turnierleitung vereinfacht, würden wir uns über eine kleine Beteiligung an diesen Kosten freuen.

Browserunterstützung

Für diese Software wird auf freie Webstandards gesetzt. Leider werden diese nicht von allen Browsern vollständig unterstützt. Deshalb kann es vorkommen, dass einzelne Funktionen mit einigen Browsern nicht zur Verfügung stehen. Um das nach Möglichkeit zu vermeiden, wird das Programm mit diesen Browsern getestet:

Desktop:

Dieser Browser unterstützt folgende Funktionen nicht:

  • JavaScript
  • HashChange
  • LocalStorage
  • WebWorkers
  • Download Attribute
  • FileReader
  • JSON
